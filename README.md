# Security Audit Project 

## Overview 
This project demonstrates a basic security audit performed on an organization. The purpose of the audit is to identify security risks, evaluate existing controls, and provide recommendations to improve the organization's security. 


## Audit Scope 
The audit reviewed: 
- User Access Management
- Password Security
- Endpoint Security
- Network Security
- Backup and Recovery
- Security Awareness


## Findings 
Key Findings identified during the audit include:
- Multi-Factor Authentication (MFA) not enabled
- Weak password requirements
- Lack of bacKup testing procedures
- Potential phishing risks

## Risk Assessment 
Risks were analyzed based on likelihood and businessimpact and documented in the risk assessment matrix.
